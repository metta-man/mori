# Mori UI Mockups V2 - 暖极简设计

## 设计方向
**暖极简 (Warm Minimalism)** - 温暖而不失简洁

### 色彩系统
```
主色 (Primary):  #FDF5E6, #FFF8E7 (米白/米黄)
强调 (Accent):   #4A4A4A, #333333 (深灰/深黑)
辅助 (Support):  #D4C5B0, #F0E6D3 (柔和棕)
```

---

## 📱 界面设计

### 1. Life Grid · 生命方格
**文件**: `life-grid-dots.html`

**特色**:
- ✨ **点阵风格 (●○)** - 用实心/空心圆点表示
- 📊 80×52 网格 = 4,160个生命方格
- 🎯 已过时间自动填充 (●)
- ⭕ 未来时间空心显示 (○)
- 💫 当前周高亮效果
- 📈 进度条可视化 (37.5%)

**设计亮点**:
- 点阵更柔和，视觉舒适
- 间距适中，整体感强
- 温暖色调减少焦虑

---

### 2. The Clock · 时钟
**文件**: `clock-countdown.html`

**特色**:
- ⏰ 大数字时钟显示 (21:30)
- ⏱️ **倒计时功能** - 距离今天结束
- 🌙 智能问候 (早上好/下午好/晚上好)
- 💬 **温暖文案** - "今天已经完成了..."
- ✨ 每秒实时更新

**设计亮点**:
- 超大字体 (72px)，一目了然
- 倒计时增加时间感
- 温暖文案提供情感支持
- 简约而不简单

---

### 3. Habit +/- · 习惯追踪
**文件**: `habit-emoji.html`

**特色**:
- 🎨 **Emoji 图标** - 每个习惯配专属emoji
  - 📚 阅读
  - 🧘 冥想
  - 🏃 运动
  - 💧 喝水
- 🔥 连续打卡天数显示
- ➕➖ 圆形按钮设计
- 🎯 大数字计数 (28px)

**设计亮点**:
- Emoji 让习惯更有个性
- 连续天数激励坚持
- 卡片式布局清晰
- Hover效果增加交互感

---

### 4. Gratitude · 感恩日记
**文件**: `gratitude-journal.html`

**特色**:
- 📅 日期显示 (2026年2月23日 · 星期日)
- 💭 **引导式提问** - "今天，有什么让你感到温暖的瞬间？"
- ✍️ 简洁输入框
- 📝 时间轴记录
- 🔥 连续记录徽章

**设计亮点**:
- 引导文案降低写作门槛
- 时间记录增加仪式感
- 圆点装饰温暖精致
- 连续记录鼓励坚持

## 🎨 设计系统

### 圆角
```
小组件: 12px
中等组件: 16px
大容器: 24-32px
```

### 阴影
```css
box-shadow: 0 8px 32px rgba(51, 51, 51, 0.08);
```

### 字重
```
标题: 300 (轻盈)
正文: 400 (适中)
强调: 400 (不粗)
```

### 间距
```
内部padding: 20-50px
卡片间距: 12-15px
元素间距: 15-25px
```

## 🎯 设计理念

1. **温暖不冷淡** - 米白色系 + 温暖文案
2. **简洁不简陋** - 功能完整 + 视觉舒适
3. **克制不冷漠** - 留白充足 + 情感细腻
4. **实用不复杂** - 功能明确 + 操作简单
