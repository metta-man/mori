### 2️⃣ 四个核心界面设计

#### ✅ Life Grid (生命方格)
- **尺寸**: 80列 × 52行 = 4,160个方格
- **含义**: 80年寿命，52周/年
- **功能**:
  - 已过时间自动填充颜色
  - 当前周高亮显示
  - 显示进度百分比 (37.5%)
- **交互**: 视觉化生命进度追踪

#### ✅ The Clock (时钟)
- **设计**: 极简圆形表盘
- **元素**: 时针、分针、中心点
- **显示**: 数字时间 (05:00)
- **特点**: 无刻度，纯净设计

#### ✅ Habit +/- (习惯追踪)
- **功能**: 简单加减计数器
- **示例习惯**: 阅读、冥想、运动
- **操作**: 点击 +/- 按钮
- **显示**: 习惯名称 + 当前计数

#### ✅ Gratitude (感恩日记)
- **输入**: 单行文本输入框
- **列表**: 历史感恩条目展示
- **示例**: 
  - "早晨的阳光透过窗帘"
  - "一杯热咖啡的温度"
  - "朋友的问候信息"

## 📊 设计系统

### 色彩应用规则
```css
/* 背景色 */
background: #FAF8F5;  /* 主背景 */

/* 文字色 */
color: #5D4E37;       /* 主要文字 */
color: #8B7355;       /* 次要文字 */
color: #B8B0A5;       /* 占位符 */

/* 强调色 */
accent: #8B7355;      /* 主要强调 */
accent: #5D4E37;      /* 深色强调 */

/* 边框 */
border: 1px solid #E8E4DE;
```

### 间距系统
- **小**: 8px
- **中**: 16px
- **大**: 24px
- **超大**: 40px

### 圆角系统
- **小组件**: 8px
- **中等组件**: 12px
- **大容器**: 16-20px
