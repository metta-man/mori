- 80列 × 52行 = 4,160格
- 代表: 80年 × 52周

- 当前周高亮

- 极简圆形设计

- 无刻度纯净

### Habit +/-

- 加减按钮
- 清晰显示

- 历史列表
- 温暖风格

## 🛠️ 定制修改

### 修改配色
找到CSS中的颜色值，替换为你需要的:

```css
/* 主背景 */
background: #FAF8F5;

/* 文字颜色 */
color: #5D4E37;

/* 强调色 */
accent: #8B7355;
```

### 修改圆角
```css
border-radius: 16px; /* 调整这个值 */
```

### 修改间距
```css
padding: 20px; /* 调整这个值 */
margin: 15px;  /* 调整这个值 */
```
