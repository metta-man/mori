Implementation Specifications

  • Input Type: Textarea with minimal diary input (3-5 sentences suggested)
  • Prompts: 5 rotating prompts (optional, user can skip)
  • Character Limit: No hard limit, but suggest 3-5 sentences (300-500 chars)
  • Timestamp: Auto-generated, date-stamped entries
  • Entry Browser: Chronological list, truncated preview (3 lines), expandable
  • Random Memory: Pull random past entry for reflection
  • Search: Full-text search across all entries (future feature)
  • Export: Export all entries as JSON/Markdown (Pro feature)
  • Privacy: Local-first storage, optional iCloud sync
