# Mori Market Sizing Framework (Draft)

## Objective
Create a defensible TAM/SAM/SOM model for Mori (memento mori + mindfulness app).

## 1) TAM (Top-down)
- Global mindfulness/mental wellness app users
- Inputs needed:
  - Global market size (USD)
  - CAGR/growth outlook
  - Mobile app share of total wellness spend

## 2) SAM (Serviceable)
- Audience segment likely to adopt reflective/mindfulness mobile routines
- Narrow by:
  - smartphone users in launch languages (EN/ZH)
  - willingness to use self-help/mindfulness apps

## 3) SOM (Obtainable, 12-24 months)
- Initial capture under realistic go-to-market assumptions
- Inputs:
  - install funnel assumptions
  - trial start + paid conversion
  - retention and churn assumptions

## Modeling Template
- Users model:
  - App installs → Activated users → Retained MAU → Paid subscribers
- Revenue model:
  - Paid users × ARPPU (monthly/annual mix)

## Next Data Requirements
1. 3+ cited sources for category size and growth
2. Regional split for EN/ZH target markets
3. Benchmark conversion/retention ranges for mindfulness apps
